Quick PDF Module

Description

Quick View PDF module enables pdf reading capability of ios applications. This module works on ipod, ipad and iphone. A quick look view at the bottom allows to switch between pages. Other functionalities like email sending, bookmarks a page and printing is available. Very efficient and easy to use. Just one line of code and you are done.

Accessing the quickpdf Module

To access this module from JavaScript, you would do the following:

var quickpdf = require("com.kuchbee.quickpdf");

The quickpdf variable is a reference to the Module object.

Functions

1- openPDF:
This function opens a pdf file from the resources directory of the project or from the document directory of the installed application.
Parameters:
It takes one dictionary parameter wich has following key-value pair values.
a) file (string): The name of the file without extension. Please don't provide '.pdf' extension in the, just name of the file.
b) password (string): If your pdf file is password protected, you can provide password to open file.
c) isMailEnabled (bool): If this is true then email option will be availabe otherwise not. d) loadFileFrom (string): If your file exist in the resources directory of the project then provide 'resources' and if your file exist in the document directory of the installed application then provide 'document' value. Currently these two value are valid and in the later version other support will be added.
e) close (callback function): If you want to get notified when pdf file is closed then this callback function will be called when pdf will be closed.

Usage

// Create a Window
var win = Ti.UI.createWindow({
backgroundColor : 'white'
});

var quickpdfreader = require('com.kuchbee.quickpdf');
Ti.API.info("module is => " + quickpdfreader);

var button = Ti.UI.createButton({ top:10, left:10, right:10, title:'Open Pdf File' });

win.add(button);
button.addEventListener('click',function(e){
quickpdfreader.openPDF({
file: 'app', password: '', loadFileFrom: 'document',
close: function(e){
alert('closed'); }
});
});

win.open();
